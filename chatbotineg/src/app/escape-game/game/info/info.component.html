Pour communiquer avec le chatbot :
  • Pour commencer, dites "salut" ou "bonjour" pour que le chatbot vous réponde.
  • Si vous êtes bloqué et que vous avez besoin d'aide, écrivez "je suis bloqué" ou "aidez-moi" ou toute autre formulation de demande d'aide.
  • Le chatbot vous posera alors la question "Pouvez-vous me dire ce que vous voyez dans la pièce ?" pour mieux comprendre votre situation.
  • Si vous ne savez pas quoi répondre, essayez de décrire ce que vous voyez dans la pièce aussi précisément que possible.
  • Si vous ne trouvez pas d'indice après avoir répondu à la première question, le chatbot vous posera une deuxième question pour obtenir plus de détails. Si vous répondez "non" à cette question, le chatbot vous donnera alors un premier indice.
  • Si vous ne trouvez toujours pas de solution après avoir suivi le premier indice, le chatbot vous proposera un deuxième indice si vous répondez "non" à la deuxième question.
  • Si vous avez besoin de nouvelles questions ou de nouveaux indices, n'hésitez pas à redemander à tout moment.
  • N'oubliez pas de continuer à explorer la pièce et à chercher des indices pour vous aider à vous échapper !

J'espère que ces instructions vous aideront à profiter de votre expérience d'escape game avec le chatbot. Bonne chance pour vous échapper de la pièce !
